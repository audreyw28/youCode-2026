# Backend

This folder is reserved for the server-side layer of the project.

Suggested responsibilities:

- expose resource data through an API
- normalize shelter, food, and community datasets
- add search, filtering, and caching logic

Starter layout:

- `src/index.js` application entrypoint
- `src/routes/` HTTP route handlers
- `src/services/` backend domain logic
- `src/data/` backend-managed data files and loaders
